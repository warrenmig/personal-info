    What is Scholarship Management System

    Specifically, design for Local Government Unit (LGU's). Developed using PHP, MySQL, and JavaScript to eliminate the need for manual data transfer from forms to database, offering a quicker and more accurate method of gathering and storing information. Scholarship Mangement System serves multiple purposes. The system provides comprehensive data on scholarships and applicants, and allows crud operation for stored information.
